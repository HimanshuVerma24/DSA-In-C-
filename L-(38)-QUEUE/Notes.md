// Queue

(1) Queue Will Follow FIFO. 

(2) It Has Front And Rear. 

(3) Data :- 
    (i) Storage
    (ii) Insert End
    (iii) Delete End

(4) Methods:- 
    (i) Enqueue
    (ii) Dequeue
    (iii) First Value
    (iv) Last Value
    (v) Peak
    (vi) isEmpty
    (vii) isFull

(5) Queue Can Be Implement By The Help Of Array,Stack,Linked-list.
